About

About

This course is part of a five-week MOOC (that you should sign up for):

Intro to R for Journalists: How to find Great Stories in Data

Who

Andrew Ba Tran is a data reporter on The Washington Post’s rapid response investigative team.

Thanks

  • Eric Bailey for creating the logo and retouching the CSS for this site
  • Robert Kabacoff for teaching me so much about teaching
  • Hadley Wickham for organizing the tidyverse and excellent tutorials that help expand the community
  • Amber Thomas, Reuben Fischer-Baum, and John Muyskens for great feedback
  • Ben Welsh for being such a great advocate for reproducible journalism

Packages and libraries

  • readr - reads rectangular data quickly
  • dplyr - grammar of data manipulation
  • tidyr - tidies data
  • ggplot2 - creates charts based on the Grammar of Graphics
  • sf - simple features for mapping
  • ggmap - Google maps features
  • leaflet - A wrapper for Leaflet.JS for interactive maps
  • datatable - Interface to the DataTables JavaScript library

Thanks to so many people for contributing to the open source package software that we use in this class and in our every day process.
